# 📱 CRUD de Produtos em Flutter

## 📌 Descrição

Este projeto consiste no desenvolvimento de um aplicativo mobile utilizando **Flutter**, com o objetivo de implementar um **CRUD completo (Create, Read, Update, Delete)** consumindo uma **API REST pública**.

A aplicação segue o padrão arquitetural proposto em aula (**Service → Provider → UI**), promovendo organização, separação de responsabilidades e facilidade de manutenção.

---

## ⚙️ Funcionalidades

O aplicativo permite:

* ➕ **Criar** novos produtos (POST)
* 📋 **Listar** produtos existentes (GET)
* ✏️ **Editar** produtos (PUT)
* 🗑️ **Excluir** produtos (DELETE)
* 🔍 **Buscar** produtos por nome

---

## 🧱 Arquitetura

O projeto foi estruturado em camadas:

* **Models** → Representação dos dados
* **Services** → Comunicação com a API (Dio)
* **Providers** → Gerenciamento de estado (ChangeNotifier)
* **UI (Screens/Widgets)** → Interface do usuário

---

## 🛠️ Tecnologias Utilizadas

* Flutter 3.x
* Provider (gerenciamento de estado)
* Dio (requisições HTTP)
* GoRouter (navegação)
